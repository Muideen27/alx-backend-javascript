# 0x02. ES6 Classes, OOP

This repository contains a collection of JavaScript files that demonstrate the use of ES6 classes and object-oriented programming (OOP) concepts.

## Resources

- [Classes Overview](https://intranet.alxswe.com/rltoken/ke2dSL31JbpAUBW0qWE9WA)
- [Metaprogramming](https://intranet.alxswe.com/rltoken/6OgF5QGbYclp_cwATfq-0g)

## Learning Objectives

This project is aimed at helping you understand the following:

- How to define a Class in JavaScript
- How to add methods to a class
- Why and how to add a static method to a class
- How to extend a class from another
- Metaprogramming and the use of symbols in JavaScript

## Requirements

- All your files will be executed on Ubuntu 18.04 LTS using NodeJS 12.11.x
- Allowed editors: vi, vim, emacs, Visual Studio Code
- All your files should end with a new line
- A `README.md` file, at the root of the project folder, is mandatory
- Your code should use the `.js` extension
- Your code will be tested using Jest, and you can run the tests with the command `npm run test`
- Your code will be verified against lint using ESLint, and you can check the entire project using `npm run full-test`
